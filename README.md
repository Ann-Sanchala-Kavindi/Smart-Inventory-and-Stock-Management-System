# AI-Based Smart Inventory Prediction System

An intelligent full-stack inventory management system that combines modern web technologies with machine learning to predict future product demand and optimize inventory operations.

This project is built using:

- React.js for the frontend
- Spring Boot for the backend REST API
- MySQL for database management
- Python (Scikit-learn) for AI/ML predictions
- Docker & Docker Compose for containerized deployment

## Features

- JWT Authentication & Role-Based Access
- Product & Inventory Management
- Sales Tracking System
- AI-Based Demand Forecasting
- Smart Restocking Recommendations
- Analytics Dashboard & Reports
- RESTful API Architecture
- Dockerized Multi-Service Environment

## Project Goal

The main objective of this project is to reduce inventory shortages and overstock issues using machine learning-based demand prediction techniques.

This project is also developed as a research-oriented software engineering solution focusing on predictive analytics and intelligent inventory optimization.
